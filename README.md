# Security Capstone

This repository is a student-level capstone project connecting my academic work, Google Cybersecurity learning, and TryHackMe lab practice.

The goal of this repository is to show how different parts of my cybersecurity learning path connect together instead of keeping them as separate, unrelated work.

---

## Purpose of This Repository

This repository answers one main question:

> How did I connect my academic coursework, cybersecurity fundamentals, and hands-on lab practice into one security-focused learning project?

This capstone is designed to connect:

- Academic cybersecurity-related coursework
- Google Cybersecurity Certificate concepts
- TryHackMe hands-on lab practice
- SOC fundamentals
- Linux and networking basics
- Log analysis and incident documentation
- Cloud & Identity Security direction
- Basic GRC-aware security thinking

---

## Capstone Focus

This project is not meant to be an advanced professional security project.

It is a student-level portfolio project designed to show:

- How I organize security knowledge
- How I connect theory with practice
- How I document technical learning
- How I relate labs to real security concepts
- How my long-term direction connects to Cloud & Identity Security
